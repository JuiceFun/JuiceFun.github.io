MySQL基础01

发表于2022-09-04|更新于2022-09-04
|阅读量:

MySql的语句：

  • 不区分大小写；
  • 以分号结尾；

1.DQL

执行顺序：

  1. from
  2. where
  3. select
  4. order by

分组函数（avg、count、min、max、sum）

  • 分组函数自动忽略null；
  • 分组函数不可以直接用在where中

    分组函数不可以在where中执行：与执行顺序有关；
    select sum(sal) from emp;//因为select 在分组查询之后执行，所以这样是可以的；

优先使用where,where无法完成时在使用having

找每个部门的平均薪资，显示平均薪资大于2000的
select avg(sal),deptno from emp group by deptno having avg(sal)>2000;
